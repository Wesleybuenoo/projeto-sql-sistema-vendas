# Sistema de Vendas — Projeto SQL (MySQL)

Projeto criado para fins de estudo utilizando MySQL.  
Inclui criação de banco, tabelas, consultas, views e procedures.

## Como usar
1. Execute o arquivo `01_create_database.sql`
2. Depois execute `02_insert_data.sql`
3. Utilize os demais para consultas e análises.

## Estrutura do projeto

├── 01_create_database.sql  
├── 02_insert_data.sql  
├── 03_queries.sql  
├── 04_views.sql
